Using the Application

Step 1: Login

  • When the application opens, the Login page is displayed.
  • Enter your User ID in the provided field.
  • Enter your Password.
  • Click the Login button to proceed.
  • The Login button becomes enabled once the User ID is entered.

Step 2: Dashboard

  • After successful login, you are redirected to the Dashboard based on your role.
  • The dashboard displays a welcome message with your user name.
  • The header and footer remain visible for easy navigation.

Step 3: Roles

  • Admin: Manages users and has access to all features.
  • Support Engineer: Manages tickets and provides support to users.
  • User: Can create tickets and view their status.

Step 4: User Manual

  • Click User Manual from the footer to access this guide.
  • This page provides a brief overview of how to use the application.

Step 5: Technical Details Learnt

  • Click Technical Details Learnt from the footer.
  • This page provides details about the project structure and technical implementation.

Step 6: Resource

  • Click Resource from the footer.
  • This page provides links to external resources and documentation.
